# 享保の改革（8代将軍：徳川吉宗）

1. 目安箱 … 広く町人の意見をきくために、設置。
2. 上げ米の制 … 幕府が大名に命じて、領地1万石につき百石の米を納めさせるかわりに、参勤交代の江戸勤めを半年に短縮。
3. 足高の制 … 有能な人材を登用する。
※ 江戸の町奉行として大岡忠相を登用。
4. 公事方御定書 … 公平な裁判の基準。
※ 新田開発（耕地面積を増やす）に力を入れる。
※ 青木昆陽に命じて、ききんに強いさつまいもを研究させる。
※ ぜいたくをいましめて、質素倹約を心がける。

# 寛政の改革（老中：松平定信）

1. 囲米の制 … ききんへの備えとして、穀物をたくわえておく。
2. 旧里帰農令 … 農村から江戸に出てきている者が村に帰ることを奨励。
3. 寛政異学の禁 … 幕府の学問所では、朱子学以外の学問を禁止。
4. 棄捐令 … 武士の借金を帳消しに。

# 天保の改革（老中：水野忠邦）

1. 株仲間の解散 … 物価を引き下げようとした。
2. 人返しの法 … 江戸に出稼ぎに出ていた農民たちに、農村に帰るよう促す。
3. 上知令 … 江戸・大阪の近くに領地を持っていた旗本や譜代大名に代わりの土地をあたえ、それを幕府の領地とする法令。（結果として失敗）
